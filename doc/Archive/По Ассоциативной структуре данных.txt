Как будет происходить тестирование и работа с новой базой данных:

1. Весь код по максимуму должен быть описан внутри базы данных, иными словами вся логика по работе с собой же, будет записана внутри себя.
2. Все внешние интерфейсы будут заниматься исключительно запуском выполнения с определённого фиксированного адреса внутри этой базы данных.
3. Соответственно с внешним интерфейсом уже и можно работать, но раз у нас есть интерфейс, значит мы получаем первый уровень абстракции. Далее заполнение базы данных можно продолжать из интерфейса, чтобы гарантировать исключение сбоев.
